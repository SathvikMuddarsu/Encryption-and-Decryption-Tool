# 🔐 Encryption & Decryption Tool (C++)

A simple C++ project that demonstrates how basic encryption works using three algorithms: Caesar Cipher, Vigenère Cipher, and RSA.

---

## What it does

* Encrypt and decrypt messages
* Lets you choose between 3 algorithms
* Runs in a simple menu-driven program

---

##  Algorithms used

* Caesar Cipher (shift-based encryption)
* Vigenère Cipher (keyword-based encryption)
* RSA (basic public-key encryption)

---

##  How to run

```bash
g++ Encryption.cpp -o app
./app
```

---

##  Why I built this

To understand how encryption works by implementing the algorithms from scratch in C++.
